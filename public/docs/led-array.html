Led Array

Run with:

node src/j5/eg/led-array.js

var five = require("johnny-five");

five.Board().on("ready", function() {
  var array = new five.Led.Array([3, 5, 6]);

  this.repl.inject({
    array: array
  });

  array.pulse();
});

Control multiple LEDs at once by creating an Led.Array. All must be on PWM pins if you want to use methods such as pulse() or fade()
