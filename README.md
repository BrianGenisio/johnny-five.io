# johnny-five.org

The repo for the johnny-five.org website

## Prerequisites

- node
- npm
- sass gem (which requires ruby)

```
brew install node;
brew install sass;
```

## Setup 

```
npm install;
```

### Build

Download and install remote content repos: 

```
grunt install
```

Regenerate all content from remote sources: 

```
grunt regen
```

Development automation: 

```
grunt && grunt dev
```
