# johnny-five.org

The repo for the johnny-five.org website

## Prerequisites

- node
- npm
- sass gem (which requires ruby)

```
brew install node;
brew install sass;
```

## Setup 

```
npm install;
```

### Build

Bootstrap site content from remote repos: 

```
grunt bootstrap
```

Regenerate all local content from remote content sources: 

```
grunt regen
```

Development automation: 

```
grunt && grunt dev
```
